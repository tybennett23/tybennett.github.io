Robotics Club: Drummer

As part of the Northwestern University Robotics Club, I helped build a drumming robot that can be programmed to play any downloaded song using Midi software. My contribution to this six-armed robot was welding the frame, to create a support system for these independent motors. I also assembled motor systems to connect each arm to a laptop. This is still a work in progess so there is more to come!
